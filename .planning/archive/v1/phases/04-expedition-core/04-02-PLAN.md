<objective>
Implement the escalation trigger system that increases threat over time and in response to player actions. This creates the time pressure that makes expeditions feel risky.
</objective>

<context>
## Prior Plan Output
- `src/expedition/escalation/escalation_manager.gd` from 04-01
- EscalationManager with escalation_level, threshold_crossed signal

## Design Reference
- `docs/design/systems/expeditions.md` — Escalation Triggers table
- Time: +1-3 per minute (configurable)
- Combat noise: +5-15 per engagement
- Loud actions: +10-25
- Sealed areas: +5-10

## Constraints
- Time escalation should pause when game is paused
- Triggers should log for debugging/tuning
- V1: single escalation profile (no per-location profiles yet)
</context>
