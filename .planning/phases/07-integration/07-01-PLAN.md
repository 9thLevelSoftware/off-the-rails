# Plan 07-01: Scene Wiring & Main Loop

<objective>
Connect the main game scene to the isometric Workshop and establish the player spawn flow. After this plan, starting a session from the main menu will load the Workshop scene and spawn the player at the correct position.
</objective>

<context>
@.planning/phases/07-integration/07-CONTEXT.md
@src/main.gd
@src/autoloads/game_state.gd
@src/train/cars/workshop/scenes/workshop.tscn
@src/train/cars/workshop/adapters/workshop_layout_controller.gd
</context>

<tasks>

## Task 1: Update main.gd Scene Path

Update the train scene path constant in main.gd to point to the isometric Workshop scene.

**Action:**
1. Open `src/main.gd`
2. Change `TRAIN_SCENE_PATH` from `"res://src/train/train.tscn"` to `"res://src/train/cars/workshop/scenes/workshop.tscn"`
3. The rest of main.gd's scene loading logic remains unchanged

**Files:** `src/main.gd`

> verification: grep -q "workshop/scenes/workshop.tscn" src/main.gd

**Done when:** main.gd references the isometric workshop scene path.

---

## Task 2: Add GameState Registration to Workshop

The Workshop scene must register itself with GameState so the player spawning mechanism works.

**Action:**
1. Open `src/train/cars/workshop/adapters/workshop_layout_controller.gd` (the workshop's main script)
2. In `_ready()`, add: `GameState.register_scene(GameState.GameScene.TRAIN, self)`
3. This triggers GameState to spawn the player when a session is active

**Files:** `src/train/cars/workshop/adapters/workshop_layout_controller.gd`

> verification: grep -q "GameState.register_scene" src/train/cars/workshop/adapters/workshop_layout_controller.gd

**Done when:** Workshop script registers with GameState on ready.

---

## Task 3: Add PlayerSpawn Marker to Workshop Scene

GameState looks for a Node2D named "PlayerSpawn" to position the player. The workshop scene needs this marker.

**Action:**
1. Open `src/train/cars/workshop/scenes/workshop.tscn`
2. Add a Node2D child named "PlayerSpawn" under the root Workshop node
3. Position it at a sensible starting location (e.g., `Vector2(200, 150)` — center-ish of the workshop floor)
4. The marker should be a direct child of the Workshop root for `get_node_or_null("PlayerSpawn")` to find it

**Files:** `src/train/cars/workshop/scenes/workshop.tscn`

> verification: grep -q 'name="PlayerSpawn"' src/train/cars/workshop/scenes/workshop.tscn

**Done when:** Workshop scene has a PlayerSpawn Node2D marker.
