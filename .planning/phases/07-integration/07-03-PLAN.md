## Task 0: Add Required Input Actions

**Action**: Add missing input actions to project.godot InputMap before implementing menus.

**PREREQUISITE**: The following input actions are required for menu functionality but do NOT exist in project.godot:
- `ui_pause` — Toggle pause menu (Escape key)

Godot's built-in `ui_cancel` can be used as fallback, but explicit `ui_pause` is cleaner.

1. Add input action via code or project settings:
   ```gdscript
   # Option A: In main.gd _ready() if action doesn't exist:
   if not InputMap.has_action("ui_pause"):
       InputMap.add_action("ui_pause")
       var event := InputEventKey.new()
       event.physical_keycode = KEY_ESCAPE
       InputMap.action_add_event("ui_pause", event)
   ```
   
   Or manually add to project.godot [input] section:
   ```ini
   ui_pause={
   "deadzone": 0.5,
   "events": [Object(InputEventKey,...,"physical_keycode":4194305,...)]
   }
   ```
   (4194305 = KEY_ESCAPE)

2. Verify action exists: `InputMap.has_action("ui_pause")` returns true

**Done when**: `ui_pause` input action exists and maps to Escape key.

## Task 3: Create Pause Menu

**Action**: Allow player to pause and access options during gameplay.

**DEPENDS ON**: Task 0 (ui_pause input action must exist)

1. Create `src/ui/menus/pause_menu.tscn` as CanvasLayer
2. Set layer to 50 (above HUD, below main menu)
3. Layout:
   ```
   PauseMenu (CanvasLayer)
   └── ColorRect (semi-transparent overlay)
       └── CenterContainer
           └── VBoxContainer
               ├── Label ("PAUSED")
               ├── ResumeButton
               └── QuitButton (return to main menu)
   ```
4. Create `src/ui/menus/pause_menu.gd`:
   ```gdscript
   func _ready() -> void:
       visible = false
       process_mode = Node.PROCESS_MODE_ALWAYS  # Process even when paused
   
   func _unhandled_input(event: InputEvent) -> void:
       if event.is_action_pressed("ui_pause"):
           toggle_pause()
   
   func toggle_pause() -> void:
       visible = !visible
       get_tree().paused = visible
   
   func _on_resume_pressed() -> void:
       toggle_pause()
   
   func _on_quit_pressed() -> void:
       get_tree().paused = false
       GameState.end_session()
       # Transition to main menu
   ```
5. Add ui_pause input action if not exists (Escape key)
6. Pause menu should be instantiated in main.gd alongside HUD

**Verify**:
- Press Escape → pause menu appears, game freezes
- Press Resume → pause menu hides, game resumes
- Press Quit → returns to main menu

**Done when**: Pause menu works correctly with proper pause/unpause behavior.
