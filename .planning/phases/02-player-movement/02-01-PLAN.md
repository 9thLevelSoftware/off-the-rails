## Task 2: Create InputConverter
**Action**: Create `src/isometric/domain/input_converter.gd` with:
- `class_name InputConverter`
- Static function `wasd_to_isometric(input: Vector2) -> Vector2` — converts WASD vector to isometric screen direction
- Uses formula: `Vector2((input.x - input.y), (input.x + input.y) * 0.5).normalized()`
- **Zero-vector handling** (critique finding: dead input):
  ```gdscript
  static func wasd_to_isometric(input: Vector2) -> Vector2:
      if input.length_squared() < 0.01:  # Dead zone check
          return Vector2.ZERO
      var iso := Vector2(input.x - input.y, (input.x + input.y) * 0.5)
      return iso.normalized()
  ```
