<objective>
Implement the additive scene architecture with Main, Train, and Expedition scenes. Verify that MCP tools can successfully create and modify scenes, establishing the MCP-driven development workflow for all future phases.
</objective>

## Target Scene Hierarchy
```
Main (persistent)
├── GameState (autoload, already registered)
├── TrainScene (always loaded, may be hidden)
├── ExpeditionScene (loaded when in field)
└── UI (persistent overlay)
```

## Additive Scene Pattern
- Main.tscn is the root scene (set as main_scene in project.godot)
- TrainScene and ExpeditionScene are loaded/unloaded additively via code
- Both scenes can coexist (for split-team play in multiplayer V2)
- GameState manages which scene is "active" for the player

## MCP Tools to Verify
- `mcp__godot-mcp__create_scene` — Create .tscn files
- `mcp__godot-mcp__add_node` — Add nodes to scenes
- `mcp__godot-mcp__modify_scene_node` — Update node properties
- `mcp__godot-mcp__read_scene` — Read scene structure
- `mcp__gdai-mcp__create_scene` — Alternative scene creation
</context>

## Task 1: Create Main Scene

**Action**: Implement `src/main.tscn` as the persistent root scene

**Instructions**:
1. Create a new scene with Node root (not Node2D/Node3D yet — game can be either)
2. Name root node "Main"
3. Add child nodes for scene attachment points:
   - `SceneContainer` (Node) — Parent for dynamically loaded scenes
   - `UILayer` (CanvasLayer) — Persistent UI overlay
4. Create `src/main.gd` with additive scene loading methods:
   ```gdscript
   extends Node

   var train_scene: Node = null
   var expedition_scene: Node = null

   func load_train() -> void:
       if train_scene == null:
           train_scene = load("res://src/train/train.tscn").instantiate()
           $SceneContainer.add_child(train_scene)

   func load_expedition(location_id: String) -> void:
       if expedition_scene == null:
           expedition_scene = load("res://src/expedition/expedition.tscn").instantiate()
           $SceneContainer.add_child(expedition_scene)
       GameState.change_location(location_id)

   func unload_expedition() -> void:
       if expedition_scene != null:
           expedition_scene.queue_free()
           expedition_scene = null
           GameState.change_location("train")
   ```
5. Attach script to Main node

**Verify**: Scene saves without errors, script has no parse errors

**Done when**: `src/main.tscn` exists with SceneContainer and UILayer children
