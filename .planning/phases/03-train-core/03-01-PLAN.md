  <name>Task 1: Create Subsystem Base Class</name>
  <files>src/train/subsystems/subsystem.gd</files>
  <action>
Create the abstract Subsystem base class that all concrete subsystems will extend.

1. Create the directory `src/train/subsystems/` if it doesn't exist.

2. Create `subsystem.gd` with the following structure:

```gdscript
class_name Subsystem
extends Node

## Abstract base class for all train subsystems.
## Provides state management and signal contracts for subsystem state changes.

## Subsystem states for V1 (Damaged/Upgraded deferred)
enum SubsystemState {
    OFFLINE,      ## Not powered or not installed
    OPERATIONAL   ## Normal function
}

## Emitted when subsystem state changes
signal state_changed(old_state: SubsystemState, new_state: SubsystemState)

## Current state of the subsystem
var current_state: SubsystemState = SubsystemState.OFFLINE:
    set(value):
        if current_state != value:
            var old = current_state
            current_state = value
            state_changed.emit(old, value)

## Display name for UI purposes
@export var display_name: String = "Subsystem"

## Whether this subsystem requires power from another subsystem
@export var requires_power: bool = false


func _ready() -> void:
    _initialize_subsystem()


## Override in subclasses for subsystem-specific initialization
func _initialize_subsystem() -> void:
    pass


## Transition to operational state
func bring_online() -> void:
    if can_go_online():
        current_state = SubsystemState.OPERATIONAL


## Transition to offline state
func take_offline() -> void:
    current_state = SubsystemState.OFFLINE


## Check if subsystem can go online (override for dependency checks)
func can_go_online() -> bool:
    return true


## Check if currently operational
func is_operational() -> bool:
    return current_state == SubsystemState.OPERATIONAL


## Get state as human-readable string
func get_state_string() -> String:
    match current_state:
        SubsystemState.OFFLINE:
            return "Offline"
        SubsystemState.OPERATIONAL:
            return "Operational"
        _:
            return "Unknown"
```

<task type="auto">
  <name>Task 2: Create Engine Subsystems (Locomotion, Power Grid)</name>
  <files>src/train/subsystems/locomotion.gd, src/train/subsystems/power_grid.gd</files>
  <action>
Create the two subsystems that belong to the Engine car.

1. Create `locomotion.gd`:

```gdscript
class_name Locomotion
extends Subsystem

## Locomotion subsystem for the Engine car.
## Controls train movement speed and fuel consumption.

## Movement speed multiplier when operational
@export var speed_multiplier: float = 1.0

## Fuel consumption rate per unit distance
@export var fuel_rate: float = 1.0


func _initialize_subsystem() -> void:
    display_name = "Locomotion"
    requires_power = true  # Needs Power Grid


## Get current speed multiplier (0 if offline)
func get_speed_multiplier() -> float:
    return speed_multiplier if is_operational() else 0.0


## Calculate fuel consumption for a given distance
func calculate_fuel_consumption(distance: float) -> float:
    if not is_operational():
        return 0.0  # Can't move, no fuel consumed
    return distance * fuel_rate
```

2. Create `power_grid.gd`:

```gdscript
class_name PowerGrid
extends Subsystem

## Power Grid subsystem for the Engine car.
## Provides power to all other subsystems on the train.

## Power output level (for future upgrade tiers)
@export var power_output: float = 100.0

## Emitted when power availability changes
signal power_availability_changed(is_available: bool)


func _initialize_subsystem() -> void:
    display_name = "Power Grid"
    requires_power = false  # This IS the power source


## Override state setter to emit power availability signal
func _set_state(value: SubsystemState) -> void:
    var was_operational = is_operational()
    current_state = value
    var is_now_operational = is_operational()
    
    if was_operational != is_now_operational:
        power_availability_changed.emit(is_now_operational)


## Check if power is currently being provided
func is_providing_power() -> bool:
    return is_operational()


## Get current power output (0 if offline)
func get_power_output() -> float:
    return power_output if is_operational() else 0.0
```

**[CRITIQUE-FIX] Signal emission timing**: Connect to state_changed in _ready() to guarantee emission on ALL state transitions:

```gdscript
func _ready() -> void:
    super._ready()
    # CRITICAL: Connect to own state_changed to guarantee power signal emission
    state_changed.connect(_on_state_changed)


func _on_state_changed(_old_state: SubsystemState, _new_state: SubsystemState) -> void:
    # CRITICAL: Always emit power signal on ANY state change
    power_availability_changed.emit(is_operational())
    print("[PowerGrid] State changed - Power available: %s" % is_operational())
```

This ensures power_availability_changed fires for BOTH bring_online() AND take_offline() transitions.

<task type="auto">
  <name>Task 3: Create Workshop Subsystem (Fabricator)</name>
  <files>src/train/subsystems/fabricator.gd</files>
  <action>
Create the Fabricator subsystem for the Workshop car.

1. Create `fabricator.gd`:

```gdscript
class_name Fabricator
extends Subsystem

## Fabricator subsystem for the Workshop car.
## Handles crafting station functionality (queue system deferred to Phase 6).

## Reference to power source for dependency check
var power_source: PowerGrid = null

## Crafting speed multiplier (for future upgrade tiers)
@export var crafting_speed: float = 1.0

## Emitted when fabricator becomes ready/not ready for crafting
signal fabricator_ready_changed(is_ready: bool)


func _initialize_subsystem() -> void:
    display_name = "Fabricator"
    requires_power = true


## Set the power source reference for dependency checking
## [CRITIQUE-FIX] Validates power_grid is not null before storing
func set_power_source(power_grid: PowerGrid) -> void:
    if power_grid == null:
        push_warning("Fabricator: Attempted to set null power source")
        return
    
    # Disconnect from old power source if exists
    if power_source and is_instance_valid(power_source) and power_source.power_availability_changed.is_connected(_on_power_changed):
        power_source.power_availability_changed.disconnect(_on_power_changed)
    
    power_source = power_grid
    
    # Connect to new power source
    if power_source:
        power_source.power_availability_changed.connect(_on_power_changed)
        # Update state based on current power
        _on_power_changed(power_source.is_providing_power())


## Handle power availability changes
func _on_power_changed(power_available: bool) -> void:
    if not power_available and is_operational():
        # Lost power - go offline
        take_offline()
        fabricator_ready_changed.emit(false)
    elif power_available and not is_operational():
        # Power restored - can potentially come online
        # Note: doesn't auto-bring online, just enables it
        pass


## Override can_go_online to check power dependency
## [CRITIQUE-FIX] Includes explicit null guard for power_source
func can_go_online() -> bool:
    if power_source == null:
        push_warning("Fabricator: No power source configured")
        return false
    if not is_instance_valid(power_source):
        push_warning("Fabricator: Power source reference is invalid")
        return false
    return power_source.is_providing_power()


## Bring online with power check
func bring_online() -> void:
    if can_go_online():
        current_state = SubsystemState.OPERATIONAL
        fabricator_ready_changed.emit(true)


## Check if ready for crafting operations
func is_ready_for_crafting() -> bool:
    return is_operational() and (power_source != null and power_source.is_providing_power())


## Get crafting speed (0 if not ready)
func get_crafting_speed() -> float:
    return crafting_speed if is_ready_for_crafting() else 0.0
```
