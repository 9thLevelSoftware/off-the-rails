## Isometric Math

Tile size: 64x32 (2:1 ratio)
Conversion: `world_x = (tile_x - tile_y) * 32`, `world_y = (tile_x + tile_y) * 16`

Distance in isometric space uses world coordinates (screen pixels), not tile coordinates. Player position is CharacterBody2D.global_position (already in world space).

## Task 3: IsometricProximityDetector (infrastructure)

**File**: `src/interaction/infrastructure/isometric_proximity_detector.gd`

**Action**:
Create infrastructure class for querying nearby interactables.

Requirements:
- `class_name IsometricProximityDetector` extends RefCounted
- Tracks registered interactables as `Array[Dictionary]` where each dict has `{id: String, position: Vector2, config: InteractableConfig}`
- `func register(id: String, position: Vector2, config: InteractableConfig) -> void`
- `func unregister(id: String) -> void`
- `func update_position(id: String, new_position: Vector2) -> void`
- `func query_in_range(from: Vector2, range_override: float = -1.0) -> Array[String]` — Returns IDs in range
- `func find_nearest_in_range(from: Vector2, range_override: float = -1.0) -> String` — Returns nearest ID or empty string
- Uses InteractionRange for distance checks
- No Node dependencies

**Verify**: `grep -q "func find_nearest_in_range" src/interaction/infrastructure/isometric_proximity_detector.gd`

**Done**: IsometricProximityDetector provides O(n) proximity queries for registered interactables.

## Task 4: InteractionStateMachine (infrastructure)

**File**: `src/interaction/infrastructure/interaction_state_machine.gd`

**Action**:
Create a state machine tracking interaction lifecycle.

Requirements:
- `class_name InteractionStateMachine` extends RefCounted
- `enum State { IDLE, IN_RANGE, INTERACTING, COOLDOWN }`
- `signal state_changed(old_state: State, new_state: State)`
- `signal entered_range(interactable_id: String)`
- `signal exited_range()`
- `signal interaction_started(interactable_id: String)`
- `signal interaction_ended(interactable_id: String)`
- `var current_state: State = State.IDLE` (readonly via getter)
- `var current_target: String = ""` (ID of interactable in range)
- `var _cooldown_remaining: float = 0.0` — Internal cooldown timer
- `var _cooldown_duration: float = 0.5` — Default cooldown
- `func update(player_pos: Vector2, detector: IsometricProximityDetector) -> void` — Called each frame
- `func tick(delta: float) -> void` — **REQUIRED**: Decrements cooldown, transitions COOLDOWN → IDLE when done
- `func try_interact() -> bool` — Attempts interaction if in range, returns success
- `func end_interaction() -> void` — Ends current interaction, starts cooldown
- Handles transitions: IDLE ↔ IN_RANGE → INTERACTING → COOLDOWN → IDLE
- **Cooldown managed internally** via tick() — caller must call every frame

**Verify**: `grep -q "signal interaction_started" src/interaction/infrastructure/interaction_state_machine.gd && grep -q "func tick" src/interaction/infrastructure/interaction_state_machine.gd`

**Done**: InteractionStateMachine manages interaction lifecycle with signals and internal cooldown via tick().
