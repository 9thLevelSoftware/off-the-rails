# MCP Tooling

The repository integrates two MCP systems for Godot development.

## Godot MCP Server

- Location: `tools/godot-mcp/`.
- Language: TypeScript.
- Runtime: Node 24.x.
- Build output: `tools/godot-mcp/build/` generated by `npm run build`.
- Validation: `python tools/validate.py --mcp`.

The server supports headless Godot operations and runtime interaction through its GDScript scripts. Runtime socket tools require the interaction server script to be installed in the Godot project.

## GDAI MCP Plugin

- Location: `addons/gdai-mcp-plugin-godot/`.
- Type: external binary GDExtension plugin.
- Autoload: `GDAIMCPRuntime`.
- Python server: `addons/gdai-mcp-plugin-godot/gdai_mcp_server.py`.

Treat the plugin as an external integration. Do not rewrite or regenerate its binary assets as part of standards work.

## Configuration

Project MCP client configuration lives in `.mcp.json`. Use `GODOT_PATH` to override local machine-specific Godot paths in scripts and CI.
